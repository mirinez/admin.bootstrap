### Projektbeschreibung

Dieses Projekt ist eine statische Administrationsoberfläche, erstellt mit HTML, CSS und Bootstrap 5 (lokal eingebunden). Ziel der Aufgabe war es, die wichtigsten Bootstrap-Komponenten praktisch anzuwenden: eine dunkle Navbar über die gesamte Breite, ein Breadcrumb zur Navigation, eine Datentabelle mit Kundendaten sowie Aktions-Buttons und eine Pagination. Eigene CSS-Anpassungen wurden in einer separaten `style.css` mit CSS Custom Properties verwaltet.
